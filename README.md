# Quiz App

## Installation

Clone the repository:
```
git clone https://mdskrumi.github.io/quiz-app/
```

Install dependencies:
```
cd quiz-app
yarn
```

## Start the development server:
```
yarn dev
```

## Admin Access
To access the admin panel, use the following credentials:

* Email: admin@email.com
* Password: admin-pass
